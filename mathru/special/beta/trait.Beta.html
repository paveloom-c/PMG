fn beta(self, y: Self) -> Self

Beta function

\Beta(x,y) = \int_0^1 t^{x-1}(1-t)^{y-1}\,dt

Fore more information: https://en.wikipedia.org/wiki/Beta_function

Arguments
  • self > 0.0
  • y > 0.0
Panics

Panics if the parameter conditions are not fulfilled.

Example
use mathru::special::beta::Beta;

let x: f64 = 0.3_f64;
let y: f64 = 0.6_f64;

let beta: f64 = x.beta(y);

fn beta_inc_reg(self, a: Self, b: Self) -> Self

Incomplete regularized beta function

I_x(a,b) = \frac{\Beta(x;\,a,b)}{\Beta(a,b)}

Fore more information: https://en.wikipedia.org/wiki/Beta_function#Incomplete_beta_function

Arguments
  • 0.0 < self < 1.0,
  • a > 0.0
  • b > 0.0
Panics

Panics if the parameter conditions are not fulfilled.

Example
use mathru::special::beta::Beta;

let a: f64 = 0.3_f64;
let b: f64 = 0.6_f64;
let x: f64 = 0.2_f64;

let beta: f64 = x.beta_inc_reg(a, b);
